# Insignias - Tagging

Fügt TTT Insignien dem ACE-Tagging Framework hinzu.

## Sprühdosen

- Schwarz -> Schwarz, Grau
- Rot -> Rod, Violett
- Blau -> Blau
- Gelb -> Gelb, Orange, Bronze
- Grün -> Grün
- Weiß -> Weiß, Silver, Platin

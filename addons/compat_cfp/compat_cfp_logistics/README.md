# Logistics

## Rucksäcke

Fügt ausgewählten Rucksäcken die `ACE_wirecutter` Funktion hinzu:

- `CFP_Cutters`

## Westen

_Benötigt ACE 3.19.4_!

Fügt ausgewählten Westen die `ACE_wirecutter` Funktion hinzu:

- `CFP_HV2_Ciras_Olive`
